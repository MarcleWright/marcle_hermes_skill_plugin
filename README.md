# marcle_hermes_skill_plugin

Marcle's personal collection of Hermes Agent skills and plugins.

## Skills

- `skills/profile-manage/custom-warm-start` — Chinese warm-start onboarding flow for personal/family Hermes profiles.
- `skills/creative/image-generation-openai` — Text-to-image and image-to-image generation via OpenAI-HK API (gpt-image-2).

## Layout

```text
skills/<category>/<skill-name>/SKILL.md
plugins/<plugin-type>/<plugin-name>/
```
